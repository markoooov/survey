# PAWS-X: A Cross-lingual Adversarial Dataset for Paraphrase Identification
PAWSをフランス語・スペイン語・ドイツ語・中国語・日本語・韓国語の6つの言語に翻訳にすることで、cross-lingualのPAWS (PAWS-X)を構築。
