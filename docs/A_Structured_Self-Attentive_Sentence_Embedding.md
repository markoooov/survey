# A Structured Self-Attentive Sentence Embedding

Self-attentionを用いた教師ありSentence embeddingを提案。Skip-Thoghtをはじめとする教師なしSentence embeddingとは異なり、文書分類等のデータセットからSentence embeddingを獲得する。文書分類タスクにおいて従来手法を上回る性能を達成。
