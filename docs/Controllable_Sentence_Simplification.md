# Controllable Sentence Simplification
テキスト平易化タスクにおいて、出力する平易さを制御する研究。制御する項目として、文長(NbChars)や入力文との編集距離(LevSim), 単語の平易度(WordRank), 構文木の深さ(DepTreeDepth)を挙げており、それらの以下のように入力文の文頭に挿入することにより、制御を行う。
