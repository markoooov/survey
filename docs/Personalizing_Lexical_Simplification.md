# Personalizing Lexical Simplification 
15人のユーザの読解レベルを調査したデータセットを用いて、語彙平易化用のデータにレベル付けを行う。  これらのデータを組み合わせて、ユーザに応じて出力を制御する語彙平易化システムを構築した。
