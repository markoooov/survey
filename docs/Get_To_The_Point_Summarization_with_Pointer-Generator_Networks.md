# Get To The Point: Summarization with Pointer-Generator Networks

生成型要約に関する論文。出力単語を"生成"するのか"入力文からコピー"するのかを切り替える機構とCoverage mechanism(過去のattention distributionを考慮する機構)を導入し、単語の繰り返しや情報の欠落などの問題を克服する。
