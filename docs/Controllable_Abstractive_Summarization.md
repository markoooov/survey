# Controllable Abstractive Summarization
要約文の長さと要約文の中心となるエンティティを制御できるモデルを提案。具体的には、要約文長と対象としたいエンティティを入力テキストの文頭にラベルとして追加する。結果として、CNN-Dailymail datasetにおいてF1-ROUGE:40.38でSoTAを達成。
