# Neural Word Embedding as Implicit Matrix Factorization
負例サンプリングを伴うskip-gramによって得られるベクトルが、PMI行列を特異値分解(SVD: *Singular value decomposition*)することによって得られるベクトルと等価であることを示した論文。
