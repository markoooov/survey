# Better Evaluation for Grammatical Error Correction

文法誤り訂正の評価に広く用いられているM2スコアを提案した論文。出力文と参照文の単語の一致度ではなく、入力文-出力文間と入力文-参照文間の編集操作に注目して一致度を計算する。また、単語単位ではなく、句レベルで計算を行う。
